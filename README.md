# -Lista de Afazeres
* Na pasta do projeto
* Criar database.sqlite na pasta database e colar seu Path no arquivo .env
*
* npm install
* composer install
* php artisan migrate
* php artisan db:seed --class=PopulateTables
* php artisan serve
* npm run dev
* usuario admin. Email: admin@admin.com, senha: admin
